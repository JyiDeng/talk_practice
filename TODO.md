# 学术口语训练系统 - 开发任务清单

## 项目概述
一款针对AI表征学习与软件工程研究人员的技术表达训练平台，通过每日学术场景练习、智能语音分析、定制化技术讨论等方式提升学术表达和技术沟通能力。

## 核心特性
- **三级难度训练体系**：Level 1（技术口语）、Level 2（研究讨论）、Level 3（技术质疑）
- **每日自动生成**：系统每天早上8点自动生成三个难度级别的新练习
- **智能练习流程**：30秒思考 + 1分钟回答，培养快速组织思路能力
- **无视频模式**：专注于文字场景描述和口语表达训练

## 技术栈
- React + TypeScript + shadcn/ui + Tailwind CSS
- Supabase（数据库 + 认证 + Edge Functions + Storage）
- 百度语音识别API
- MiniMax语音合成API + 对话API
- Aliyun Qwen API（用户自备，可选）

## 系统特性

### 三种难度模板

#### Level 1 - 渐进难度口语训练
- 技术解释：解释概念、模型设计、系统模块
- 工程讨论：讨论系统设计、模型选择、性能问题
- 技术决策：在多个方案中做决策并说明理由
- 回答时间：30秒-2分钟

#### Level 2 - 研究型口语训练
- 研究组会讨论
- 论文brainstorming
- 实验失败分析
- 模型设计讨论
- 研究方向探索
- 回答时间：1-3分钟

#### Level 3 - 高压面试/技术质疑训练
- 技术面试
- 系统设计评审
- 论文答辩
- 实验方法质疑
- 模型设计挑战
- 回答时间：1-3分钟

### 练习流程
1. 用户选择难度级别（Level 1/2/3）
2. 阅读场景描述和关键要点
3. **思考30秒**（可跳过）
4. **录音1分钟**（建议时长）
5. 系统自动语音识别和AI分析
6. 查看详细分析报告

### 每日任务生成
- 系统每天早上8点自动调用`daily-task-generator` Edge Function
- 为每个难度级别生成一个新任务
- 任务存储在`practice_tasks`表中
- 用户可以在练习页面查看和选择

## 注意事项

### 定时任务配置
需要配置定时任务每天早上8点调用`daily-task-generator` Edge Function。

可选方案：
1. **Supabase pg_cron扩展**（推荐）
2. **GitHub Actions定时任务**
3. **Vercel Cron Jobs**
4. **外部定时任务服务**

### 用户使用说明
1. 首次使用：注册账户（第一个用户自动成为管理员）
2. 每日练习：选择难度级别，完成思考和录音
3. 查看分析：了解各维度表现和改进建议
4. 周报分析：每周一生成上周表现报告
5. 即时对练：与AI导师实时对话练习

### 管理员功能
- 查看所有用户信息
- 管理用户角色
- 查看所有练习记录和分析结果
