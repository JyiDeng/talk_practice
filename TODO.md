# 职场表达力提升系统 - 开发任务清单

## 项目概述
一款针对职场人士的中文和英文表达能力训练平台，通过每日职场案例练习、智能语音分析、定制化场景模拟等方式提升职场沟通能力。

## 技术栈
- React + TypeScript + shadcn/ui + Tailwind CSS
- Supabase（数据库 + 认证 + Edge Functions + Storage）
- 百度语音识别API
- 百度/MiniMax语音合成API
- Aliyun Qwen API（用户自备）+ MiniMax对话API（备选）

## 用户使用说明

### 首次使用
1. 访问系统，点击"注册"创建账户
2. 第一个注册的用户将自动成为管理员
3. 完善个人设置中的职业背景信息

### 每日练习
1. 在首页查看今日推荐任务
2. 点击"开始练习"进入练习页面
3. 观看视频后，点击"开始录音"
4. 用1-2分钟概括视频内容
5. 系统自动进行语音识别和AI分析
6. 查看详细的分析报告和改进建议

### 查看历史
1. 在"练习历史"页面查看所有练习记录
2. 点击任意记录查看详细的分析结果

### 周报分析
1. 在"周报分析"页面点击"生成上周周报"
2. 查看雷达图和各维度得分
3. 了解自己的进步轨迹

### 场景模拟
1. 在"场景模拟"页面选择场景类型
2. 系统根据您的职业背景生成定制化场景
3. 查看场景描述、挑战点和评估标准

### 即时对练
1. 在"即时对练"页面与虚拟导师对话
2. 获得实时反馈和建议
3. 练习各种职场沟通场景

## 管理员功能
- 查看所有用户信息
- 管理用户角色
- 查看所有练习记录和分析结果
